# Client Portal (Practice)

## Menu (MVP)
- Dashboard
- Patients
  - Leads
  - Assessments
  - Appointments
- Availability
- Sales
- Settings
- Help & Support (optional activation)

## Patients rules
- Leads: all contact captures
- Assessments: subset with assessment completed
- Appointments: subset with appointment scheduled/completed

Conditional Report button:
- Shown only if assessment exists

## Sales
- Appointment list
- Set show/no-show/pending
- Set won/lost/pending
- If lost, objection required
